## Калькулятор краски

### Задание
Напишите программу для подсчета, сколько банок краски нужно для определеннйо площади.

### Условия

Одной банки краски хватает на 5 кв.метров.

### Пример

```
Ширина стены (м): 5.2
Высота стены (м): 3.4
Вам понадобится 4 банок краски.
```
