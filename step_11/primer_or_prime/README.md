## Чекер простых чисел 

### Задание

Написать программу, которая принимает от пользователя число и проверяет является ли оно простым.

### Условия

Нет обязательных условий.

### Пример
```
Введите число:
>>> 4
Это число не является простым
```
```
Введите число:
>>> 2
Это простое число
```
```
Введите число:
>>> 1
Это число не является простым
```
