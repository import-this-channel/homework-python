## Сколько это времени

### Задание

Написать программу, которая считает сколько часов, минут, секунд в указанном перидиоде времени (в днях).

### Условия

Нет обязательных условий.

### Пример ввода

```
Введите количество дней:
>>> 24 
```

### Пример вывода

```
В 24 днях:
576 часов,
34560 минут,
20730600 секунд.
```
