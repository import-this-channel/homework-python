## Генератор названия рок-группы

### Задание

Написать программу, которая собирает название группы из ответов пользователя

### Условия

Нет обязательных условий

### Пример ввода

```
Ваш любимый цвет?
>>> Желтый
Ваше любимое животное?
>>> Лабрадор
```


### Пример вывода

```
Ваша группа могла бы называться "Желтый Лабрадор"
```
