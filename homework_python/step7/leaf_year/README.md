## Високосный год

### Задание

Написать программу, которая проверяет введенный пользователем год.
И выводит ответ, високосный это год или нет.

### Условия

Нет обязательных условий.

### Пример

```
Введите год:
>>> 1600
1600 это високосный год
```

```
Введите год:
>>> 2200
2200 это невисокосный год
```
